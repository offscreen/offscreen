Note Before Upload Your Book to Translate

Class No Translate

In Bracket, cinema (with one space after the word) replace with <span class="notranslate" translate="no">cinema</span> (with one space after span tag).

For sensitive text: h1-h6, div, p, span, etc, for example on film and novel title, add class="notranslate" translate="no" with specific HTML tag.

Example

<h2>Gaze</h2> (before)

<h2 class="notranslate" translate="no">Gaze</h2> (after)

From .pdf to .doc (Without Hypen)

For ^p, this note:

  1. Select text on Portable Document Format file.
  2. Press Ctrl+C.
  3. Go to rich text editor on Microsoft Word.
  4. Press Ctrl+V.
  5. Mark with === on any paragraph as in the original text.
  6. Press Ctrl+A.
  7. Press Ctrl+H.
  8. See Find & Replace form, in Find what input, type ^p (paragraph mark).
  9. See Find & Replace form, in replace whit input, type (space bar) only once.
  10. Click Replace All.
  11. Then: See Find & Replace form, in Find what input, type === (with one space bar).
  12. See Find & Replace form, in replace whit input, type ^p (paragraph mark) only once.
  13. Click Replace All.

From .pdf to .doc (With Hypen)

For -^p, this note:

  1. Select text on Portable Document Format file.
  2. Press Ctrl+C.
  3. Go to rich text editor on Microsoft Word.
  4. Press Ctrl+V.
  5. Mark with === on any paragraph as in the original text.
  6. Press Ctrl+A.
  7. Press Ctrl+H.
  8. See Find & Replace form, in Find what input, type -^p (hyphen with paragraph mark).
  9. See Find & Replace form, in replace whit input, type ^p (paragraph mark) only once.
  10. Click Replace All.
  11. And, see your document, mark with === on any paragraph as in the original text.
  12. Press Ctrl+A.
  13. Press Ctrl+H.
  14. Then. See Find & Replace form, in Find what input, type ^p (paragraph mark).
  15. See Find & Replace form, in replace whit input, type (space bar) only once.
  16. Click Replace All.
  17. And, then. See Find & Replace form, in Find what input, type === (with one space bar).
  18. See Find & Replace form, in replace whit input, type ^p (paragraph mark) only once.
  19. Click Replace All.

Convert .doc to HTML5

The key is ^&: as Italic or Bold value.

To Italic

  1. Ctrl+A
  2. Ctrl+H
  3. Find > Format > Font > Font style > Italic
  4. Replace with > <em>^&</em>
  5. Replace All

CAUTION to normalize:

  1. Ctrl+A
  2. Ctrl+H
  3. Find > Format > Font > Font style > Regular

To Bold

  1. Ctrl+A
  2. Ctrl+H
  3. Find > Format > Font > Font style > Bold
  4. Replace with > <strong>^&</strong>
  5. Replace All

CAUTION to normalize:

  1. Ctrl+A
  2. Ctrl+H
  3. Find > Format > Font > Font style > Regular

To Paragraph

  1. Ctrl+A
  2. Ctrl+H
  3. Find > Format > Font > Font style > Regular
  4. Find > ^p
  5. Replace with > </p>^p<p>
  6. Hit Ctrl+I twice (with the cursor still in the Replace with box)
  7. Hit Ctrl+B twice (with the cursor still in the Replace with box)
  8. To make sure your paragraph tags don't inherit any bold/italic formatting from the previous text.
  9. Replace All
  10. Manually type an opening tag <p> at the beginning of the first paragraph.
  11. Manually delete one from the end of the final paragraph.
